## 기능 설명

일차항의 계수 == b
상수항 == c
계수와 상수항을 순차로 입력하면 **인수분해** 시행

※ 정수 범위에서만 동작

### 예시 입출력
예시 입력
```
45 500
```
예시 출력
```
x + 25 , x + 20
x + 20 , x + 25
```

## 자세한 설명

모든 동작은 c != 0 일때 시행된다.

c > 0일때 -c에서 c까지 for문 시행  
c < 0일때 c에서 -c까지 for문 시행  

i가 c를 나누어떨어지게 하며 c/i 와 i의 합이 b와 같을 때 시행  
동작은 c//i와 i의 부호에 따라 결정된다  

(위 서술대로 순차 + + 일때)  
`"x", "+" c//i, "," "x", "+", i` 순차로 출력  

(위 서술대로 순차 + - 일때)   
`"x", "+" c//i, "," "x", i` 순차로 출력  

(위 서술대로 순차 - + 일때)  
`"x", c//i, "x", "," "+" i` 순차로 출력  

(위 서술대로 순차 + + 일때)  
`"x", c//i, "x", "," i` 순차로 출력  

다음과 같이 4부류로 동작하는 이유:음수는 부호와 함께 서술되기 때문이다.
서로 다른 두 정수로 인수분해될때는 2번 출력된다
서로 같은 두 정수로 인수분해될때는 1번 출력된다
